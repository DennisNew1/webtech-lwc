# webtech-lwc

## Vorrausetzungen
- IDE z.B. VS Code
- Node.js (Am Besten aktuelle LTS-Version => 16.15.1)

## Installatation & Entwicklung

```bash
git clone https://github.com/DennisNew1/webtech-lwc
cd webtech-lwc/
npm install
npm run watch
```

Nun sollte der Server erreichbar sein unter `http://localsthost:3001`
